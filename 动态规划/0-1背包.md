## 0-1背包问题
第i个物品的体积和价值分别为w[i],v[i],求在总体积不超过capability的前提下，所能达到的最大总价值

### 思路
考虑每个物品选或者不选
dfs(i,c)=max(dfs(i-1,c),dfs(i-1,c-w[i])+v[i])

### 代码
```py
def zero_one_knapsack(capability,w,v):
  n=len(w)
  @cache
  def dfs(i,c):
    if i < 0:
      return 0
    if c < w[i]:
      return dfs(i-1,c)
    return max(dfs(i-1,c),dfs(i-1,c-w[i])+v[i])

  return dfs(n-1,capability)
```
